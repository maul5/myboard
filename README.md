# 내게시판 만들기

## 개요
- 목적 : 게시판 만들기로 숙련도 높이기
- 사용 기술 :
Bootstrap 3.3.6
jQuery 3.0.0
Backbone 1.3.3
JSP 2.2
Servlet 3.0.1
JSTL 1.2
Spring 4.1.0 RELEASE
JPA 1.8.0 RELEASE
Java 1.8
MariaDB 5.5.49
Ubuntu 14.04.1

## 구현할 기능 목록
01. 게시글 작성하기
02. 게시글 목록보기
03. 게시글 상세보기
04. 게시글 수정하기
05. 게시글 삭제하기
06. 게시글 목록 페이징처리
07. 댓글 작성
08. 댓글 수정
09. 댓글 삭제
10. 파일 업로드
11. 로그인/로그아웃
